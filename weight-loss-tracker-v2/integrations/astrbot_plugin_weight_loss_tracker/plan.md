# 减肥追踪 AstrBot 插件计划

## MVP

- 从 `aiocqhttp` 事件读取发送者 QQ 号并自动解析后端用户
- 提供资料、食物、运动、体重、每日汇总、趋势和报表 LLM tools
- 对 LLM 估算的营养和运动消耗执行预览确认
- 支持取消待确认操作和撤销本次插件会话中的最近写入
- 通过 HTTP 调用 V2 后端，不维护插件数据库
- 提供 pytest、导入检查和真实容器 smoke test

## 增强项

- 增加更多确定性指令和记录修正能力
- 支持一条消息拆分多个食物条目
- 将待确认操作持久化到后端
- 增加图片食物识别

## 未来项

- 营养数据库校准
- 体脂秤和健康 App 数据源适配器
- 非 QQ 平台身份映射

当前阶段：MVP 已完成，等待随 V2 Docker 服务进行真实运行部署验证。
